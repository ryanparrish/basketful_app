ℹ️ How to Create and Download a Combined Order

  1. Specify the order details: Fill in the program and select the orders to include in this combined order.
  2. Save the combined order: Click the "Save" button to create the combined order.
  3. Return to the list page: After saving, navigate back to the Combined Orders list page.
  4. Download the PDF: Select your newly created combined order from the list and use the "Download Combined Order PDF" action from the actions dropdown.
